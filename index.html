Notable Findings

Missing Security Headers — CSP, HSTS, X-Frame-Options High
blog.mygov.in · CWE-693 · OWASP A05:2021

Multiple critical HTTP security headers absent or misconfigured, exposing the platform to clickjacking, MIME sniffing, and injection attacks.

Cookie Security Flags Missing (HttpOnly, SameSite, Secure) Medium
blog.mygov.in · CWE-1004 · OWASP A02:2021

Session cookies issued without HttpOnly and SameSite flags, increasing risk of session hijacking via XSS and CSRF attacks.

Swagger UI Publicly Accessible Medium
developers.mygov.in · CWE-200 · OWASP A01:2021

Swagger UI exposed without authentication, disclosing full API schema and endpoint structure to unauthenticated users.

Invalid CORS Wildcard Origin (*.mygov.in) Medium
mygov.in · CWE-942 · OWASP A05:2021

CORS policy uses wildcard subdomain pattern not supported by browsers, creating false security assumptions and potential subdomain takeover risk.

Business Logic — AI-Assisted Quiz Answer Retrieval Low
quiz.mygov.in · OWASP A04:2021 — Insecure Design

Quiz platform lacks screen capture restrictions (FLAG_SECURE), allowing AI tools to extract and answer questions, undermining quiz integrity.

Information Disclosure via HTTP Response Headers Info
mygov.in subdomains · CWE-200

Server version, framework details, and technology stack exposed through response headers, aiding attacker reconnaissance.
